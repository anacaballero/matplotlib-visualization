# Matplotlib & Seaborn Visualization Reference

Complete working reference for matplotlib and Seaborn visualization, covering the full Chapter 4 of the Python Data Science Handbook (VanderPlas). Each notebook is a self-contained, runnable guide to a specific visualization technique.
